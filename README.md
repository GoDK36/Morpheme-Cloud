# Morpheme-Cloud

1. 받은 텍스트를 konlpy의 Kkma를 이용하여 각 형태소 별로 태깅을 해줍니다.
2. 태깅된 텍스트를 [N,V,A,D]로 시작되는 내용어 리스트를 생성.
3. 워드 클라우드, matplotlib을 이용하여 폰트와 사이즈 등을 설정하여 보여줍니다.
